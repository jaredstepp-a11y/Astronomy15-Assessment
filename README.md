# 🌌 Astronomy 15 — Universe Assessment

An interactive web-based assessment aligned with the **Alberta Astronomy 15** curriculum.

## Topics Covered

| Module | Topic |
|--------|-------|
| 💥 | Big Bang Theory |
| ⭐ | Stellar Evolution |
| 🌌 | Size of the Universe |
| 🌈 | Spectrum & Colors |
| 🕳️ | Dark Matter & Dark Energy |

## Features

- 5 interactive learning modules (overview → key concepts → reference material)
- 25 multiple choice questions with immediate feedback and explanations
- Score tracking per module and overall grade
- Space-themed design with animated star field

## Usage

Open `index.html` in any modern web browser — no installation or build step required.
